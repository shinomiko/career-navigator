# Карьерный навигатор

Веб-ресурс для выбора профессии и построения карьеры.

## Функции
- Регистрация и вход (JWT)
- Три роли: абитуриент, вуз, работодатель
- Вакансии и образовательные программы
- Профориентационный тест с рекомендациями
- Карьерный план

## Запуск
```bash
npm install
node server.js
